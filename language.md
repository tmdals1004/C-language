# 자료형
```C
int -> 정수형
float -> 실수형
double -> 실수형(float<double)
char -> 문자열
```
# 배열
```C
뜻 -> 어떤 한 자료형이 연속적으로 나열된 것
선언 -> 자료형 변수이름[갯수];
초기화 -> 자료형 변수이름[갯수] = {요소};
```
# 함수
```C
이용목적 -> 어떤 특수한 목적을 이루기위해 만들어진 코드
이용이유 -> 반복적인 프로그래밍 피함
선언 -> 반환자료형 함수이름(매개변수){코드}
사용방법 -> 1. 먼저 선언한후, 아래서 코드작성(이 방법 사용시 선언할때 끝에 세미콜론 붙임)
           2. 선언과 동시 작성후 사용
재귀함수 -> 말그대로 함수안에서 함수호출(컴파일 시간이 오래걸림)
```
# for문
```C
이용목적 -> 특정 코드를 몇회 반복 필요시 사용
사용법(선언) -> for(int i=x;i<y;i++/i--)
               i가 이미 선언이 되있다면 int는 안써도 됨
중첩반복문 -> for문 안에 for문이 쓰여짐
```
# while 문
```C
이용목적 -> 특정 조건에 맞춰 코드를 반복 필요시 사용
사용법(선언) -> while (True){코드} 
               True는 무한반복이기에 중간에 break나 조건식을 같이 써줌
```
## for문과 while문의 차이점
```
for문은 몇회 반복할지 정해져있지만, while문은 조건에 맞춰 반복횟수가 정해짐
```
# 기억 클래스
```C
auto -> 코드 실행때만 일시적으로 생성(메모리 stack 저장)
static -> 선언한후 그 코드안에서 계속 실행(지역변수,현재파일에서만 사용,데이터형 저장)
extern -> static과 비슷하지만 다른파일간에서의 공유가 가능하다(데이터형으로 저장)
register -> auto형과 비슷하지만 저장공간이 CPU의 register에 저장된다
```
# 라이브러리 함수
```C
문자형
int isdigit(int c) -> c가 숫자인지 검사후, 숫자면 참값을 반환
int islower(int c) -> c가 소문자이면 참값을 반환
int isupper(int c) -> c가 대문자이면 참값을 반환
int tolower(int c) -> c를 소문자로 변환
int toupper(int c) -> c를 대문자로 변환
숫자형
int isdigit(int c) -> 문자 c가 숫자이면 참값을 반환
```
# 포인터
```C
뜻 -> 메모리의 주솟값을 저장하는 변수
포인터 연산자 -> 주소의미 - &변수이름
                참조연산자 - *변수이름
선언 -> 자료형 *포인터 이름
```

```C
뜻 -> 하나의 이름안에 서로다른 데이터형을 갖는 변수들이 저장되있는 공간
```
